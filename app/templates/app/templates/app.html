Foot Guard is here!

You now have

  • RequireJS (http://requirejs.org/)
  • Backbone (http://backbonejs.org/)
  • Sass (http://sass-lang.com/)
  • HTML5 Boilerplate (http://html5boilerplate.com/)
  • Twitter Bootstrap (http://twitter.github.com/bootstrap/)
  • Mocha test (http://visionmedia.github.com/mocha/)
  • with Test'em (https://github.com/airportyh/testem)

installed.

Generators

  • yo footguard:collection create Backbone Collection class (with model if you want)
  • yo footguard:model create Backbone Model class
  • yo footguard:helper create a helper class
  • yo footguard:view create Backbone View class (with model, template and Sass file)

Command

  • grunt server to launch your project with watch and livereload
  • grunt compile to compile all Sass, less and CoffeeScript files
  • grunt build to build a deploy version of your project
  • testem to launch unit test with Test'em'

For start

Delete this HTML code in app/templates/app.html and Sass style in src/sass/_app.sass.
Start coding!
